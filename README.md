# TEE 10A — WordPress Theme суулгах гарын авлага

Сайн байна, багшаа! Та доорх алхамуудыг дагаад theme-оо суулгаж, ангийнхаа танилцуулга сайтыг бэлэн болгоно.

---

## 1. Theme-г WordPress-д суулгах

1. WordPress admin руу нэвтрэн орно: `https://тань-домэйн.mn/wp-admin`
2. Зүүн талын цэснээс **Appearance → Themes → Add New** дарна.
3. Дээд талд **Upload Theme** товчийг дарна.
4. **Choose File** дарж `tee-10a.zip` файлыг сонгоно.
5. **Install Now** дарна.
6. Суулгасны дараа **Activate** дарж идэвхжүүлнэ.

> Theme идэвхжсэний дараа сайдбараас **Багш нар**, **Сурагчид**, **Бүтээлүүд**, **Дурсамжууд** гэсэн шинэ цэснүүд гарч ирнэ. Энэ нь theme-ийн Custom Post Types.

---

## 2. Үндсэн хуудаснуудыг үүсгэх

Theme-ийн `front-page.php` нь Нүүр хуудсыг өөрөө бариад үзүүлдэг. Бусад 4 хуудсыг гараар үүсгэнэ:

1. **Pages → Add New** руу очно.
2. Дараах 4 хуудсыг тус бүрд нь үүсгэнэ:

| Гарчиг | Slug | Template (баруун талын Page Attributes доор) |
|---|---|---|
| Багш нар | `teachers` | **Багш нар** |
| Сурагчид | `students` | **Сурагчид** |
| Хичээл & Бүтээл | `projects` | **Хичээл & Бүтээл** |
| Дурсамж | `gallery` | **Дурсамж** |

> Хуудас бүрт **Content хоосон үлдээж болно** — template нь автоматаар CPT-уудаа татаж харуулна.

3. **Settings → Reading** руу орж:
   - **Your homepage displays** → *A static page*
   - **Homepage** → шинээр үүсгэсэн `Нүүр` хуудас (эсвэл хоосон хуудас үүсгээд `Home` гэж нэрлэ).

> Дашрамд хэлэхэд, **front-page.php** нь *static homepage*-ийн агуулгыг үл хамааран өөрөө автоматаар Hero + Багш/Сурагч/Бүтээл/Дурсамжийн preview-уудыг харуулдаг.

---

## 3. Цэс (menu) үүсгэх

1. **Appearance → Menus → Create a new menu** руу очно.
2. Нэр өгнө: `Үндсэн цэс`.
3. Зүүн талаас 4 хуудсаа сонгож, **Add to Menu** дарна.
4. Доор **Display location → Үндсэн цэс** хайрцагт чагт тавьж **Save Menu** дарна.

---

## 4. Сургуулийн лого солих (заавал биш)

Theme нь өөрийн default лого (SVG) ашигладаг. Хэрвээ сургуулийн PNG/JPG логыг ашиглахыг хүсвэл:

1. **Appearance → Customize → Site Identity** руу очно.
2. **Select logo** → лого upload хийнэ.
3. **Publish** дарна.

---

## 5. Hero текст, харилцах мэдээлэл өөрчлөх

1. **Appearance → Customize** руу очно.
2. **Нүүр хуудас — Hero** — Hero дээд бичиг, гарчиг, тайлбар.
3. **Хөл хэсэг** — хаяг, и-мэйл, утас.

---

## 6. Багш нар оруулах (2 багш)

1. Зүүн талын цэснээс **Багш нар → Шинэ багш**.
2. Гарчигт багшийн нэрийг бичнэ (жишээ: *Б. Энхтуяа*).
3. Контент талбарт богино намтар бичнэ.
4. Баруун талд **Featured image** дарж зураг оруулна (нийтлэг 600×600 px, jpg/png).
5. Доор **Багшийн мэдээлэл** meta хайрцагт:
   - Заадаг хичээл (жнь: *Программчлал*)
   - Албан тушаал / үүрэг (жнь: *Анги удирдсан багш*)
   - И-мэйл хаяг
   - Уриа үг (товч quote)
6. **Publish** дарна.
7. 2 дахь багшийн хувьд дээрх алхмыг давтана.

---

## 7. Сурагчид оруулах (9 сурагч)

1. **Сурагчид → Шинэ сурагч**.
2. Гарчигт сурагчийн нэрийг бичнэ.
3. Контентэд богино танилцуулга (хоосон үлдээж болно).
4. **Featured image** оруулна.
5. **Сурагчийн мэдээлэл** хайрцагт:
   - Үүрэг (заавал биш — *Дарга*, *Нярав* г.м.)
   - Сонирхол: таслалаар тусгаарласан жагсаалт (жнь: *Кодчилол, Шатар, Хөгжим*)
   - Уриа үг
6. **Publish**. Бусдыг давтана.

> 💡 Зургийг шууд оруулахаас өмнө **Media → Add new** руу 9 сурагчийн зургийг бүгдийг нь нэг дор upload хийчихвэл хурдан болно.

> 💡 Сурагчдын дарааллыг сонгохдоо баруун талын **Page Attributes → Order** талбарт 1, 2, 3... оруулна.

---

## 8. Бүтээлүүд оруулах

1. **Бүтээлүүд → Шинэ бүтээл**.
2. Гарчигт бүтээлийн нэр (жнь: *Smart Garden — Arduino төсөл*).
3. Контентэд дэлгэрэнгүй тайлбар.
4. **Featured image** оруулна (800×600 px тохиромжтой).
5. **Бүтээлийн мэдээлэл** хайрцагт:
   - Хичээл (жнь: *Электроникийн үндэс*)
   - Багийн гишүүд (жнь: *Б. Анар, Г. Тэмүүлэн*)
   - Гадаад холбоос (GitHub эсвэл video link, заавал биш)
6. **Хичээлүүд** taxonomy хэсэгт хичээлийн нэр нэмж сонгоно (жнь: *Веб хөгжүүлэлт*, *Робот*) — энэ нь Бүтээл хуудсан дээрх шүүлтүүрт ашиглагдана.
7. **Publish**.

---

## 9. Дурсамжит зургуудыг (Memories) оруулах

Таны фолдерт аль хэдийн `memory-1.jpg`...`memory-4.jpg`, `achievement-1.jpg`...`achievement-3.jpg` зургууд байна. Тэдгээрийг:

1. **Media → Add New** руу очиж бүгдийг нь upload хийнэ.
2. Дараа нь **Дурсамжууд → Шинэ дурсамж** дарж:
   - Гарчиг (жнь: *Бямбадорж — Робот тэмцээний шагнал*)
   - **Featured image** дээр зургаа сонгоно.
   - **Дурсамжийн мэдээлэл** хайрцагт огноо, газар нэмж болно.
   - **Publish**.

---

## 10. Зөвлөмж / Гажиглах

- **Permalink асуудал** гарвал **Settings → Permalinks → Save Changes** дарна. Энэ нь rewrite rules-ыг шинэчилнэ.
- **Зургийн хэмжээ**: Theme нь автоматаар 600×600 (профайл), 800×600 (бүтээл), 1200×800 (дурсамж) гэсэн хэмжээгээр зургийг үүсгэдэг.
- **Хичээлүүд filter** Бүтээл хуудсан дээр харагдахын тулд хамгийн багадаа 1 бүтээлд `tee_course` taxonomy нэмэгдсэн байх ёстой.

---

## 11. Дизайны өнгөнүүд

Манай theme сургуулийн логоны 4 brand color дээр суурилсан:

| Өнгө | HEX | Хаана харагдах |
|---|---|---|
| 🟡 Шар | `#FFB400` | Багш нар, accent, hero dot |
| 🔴 Улаан | `#E63946` | Сурагчид, beak shape |
| 🟢 Ногоон | `#2BA84A` | Бүтээлүүд, tail |
| 🔵 Цэнхэр | `#3B7DD8` | Дурсамж, link color |

Үндсэн фон: Цэвэр цагаан (`#FFFFFF`).
Зөөлөн фон: Cream (`#FAF7F0`).
Бараан хэсэг: Бараан хар (`#0A0A0A`).
